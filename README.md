# analise-financeira-py
Atividade Extensionista - Análise e Desenvolvimento de Sistemas
Sistema desktop de educação financeira desenvolvido em Python com interface gráfica (CustomTkinter), criado como atividade extensionista do curso de Análise e Desenvolvimento de Sistemas.

----
## Telas

### Menu Principal


Tela inicial do sistema. Apresenta o nome do projeto e os três módulos disponíveis: Orçamento, Cartão de Crédito e Simular Compra. O acesso a cada funcionalidade é feito pelos botões centrais.

------

### Orçamento — Regra 50-30-20

O usuário informa sua renda mensal e o sistema calcula automaticamente a divisão recomendada com base na regra 50-30-20, método criado pela senadora americana Elizabeth Warren. https://einvestidor.estadao.com.br/colunas/luis-claudio-agora-investimentos/metodo-50-30-20-elizabeth-warren/

------

### Cartão de Crédito
 
 
O usuário informa sua renda e o limite do cartão de crédito. O sistema calcula qual porcentagem da renda o limite representa e classifica a situação em três categorias.

------

### Simular Compra
 
 
O usuário informa sua renda e o valor do produto que deseja comprar. O sistema calcula o número de parcelas recomendado para que o valor da prestação não ultrapasse 30% da renda mensal, evitando comprometimento excessivo do orçamento.

------

## Tecnologias utilizadas
 
- Python 3
- CustomTkinter
